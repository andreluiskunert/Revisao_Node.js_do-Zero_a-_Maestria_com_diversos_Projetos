Criando uma API com Express
●
Para criar a API teremos uma tarefa bem simples, comparada a estrutura
com Handlebars;
●Basta instalar o Express, ele fará tudo sozinho neste ponto;
●Depois criaremos uma rota que responde em JSON, este é o dado de
comunicação entre aplicação e API;
●É importante definir o verbo correto, como GET ou POST, por ex;
●Vamos ver na prática!
