Verbos HTTP
●
Os verbos HTTP andam de mãos dadas com as APIs, e fazem parte do
REST;
●Temos métodos como: GET, POST, PUT, DELETE, PATCH e etc;
●Cada um representa uma ação na API, ou seja, são essenciais;
●É muito importante que as nossas funcionalidades usem os métodos
corretos;
●
Enviamos o método através da requisição;
